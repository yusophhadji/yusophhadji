About Me
Make it work, make it right, make it fast.

When I started learning to code, I would often get stuck. I remember when basic array methods or loops won’t make sense to me. I would get stuck on problems about what to use and where.

That said, we all went through that difficult early phase of learning and getting started. If you’re frustrated only because you don’t think you’re there yet skill-wise, but really want to be able to answer that call, stick with it. It will come. Learn everything you can. The first time you have that exciting moment, when a big piece of the puzzle clicks for you, you’ll know whether it’s something you can be passionate about.

Only you can answer what motivates you: there’s no shame in being motivated by compensation. But if it’s not in your blood, it will indeed be “just a job,” and can be a difficult one at that. At that point you’ll just have to decide if it’s worth it.
